Pokud chceš nahrát svůj web na internet zdarma, doporučuji ti použít GitHub Pages. Zde je návod, jak to můžeš udělat:

Krok 1: Vytvoření GitHub účtu

  1. Pokud ještě nemáš účet na GitHubu, přihlas se na https://github.com a vytvoř si účet.

  2. Pokud účet máš, přihlas se.

Krok 2: Vytvoření nového repozitáře

  1. Po přihlášení klikni na New (Nový repozitář).

  2. Zadej název repozitáře, například hyx-website.

  3. Zaškrtněte možnost Public.

  4. Klikni na tlačítko Create repository.

Krok 3: Nahrání souborů (HTML a CSS) do repozitáře

  1. Po vytvoření repozitáře přejdi na stránku repozitáře.

  2. Klikni na Add file a poté na Upload files.

  3. Nahraj soubory index.html a style.css, které jsme vytvořili dříve.

  4. Klikni na Commit changes.

Krok 4: Aktivace GitHub Pages

  1. V repozitáři klikni na Settings (Nastavení).

  2. Sjeď dolů, kde je sekce GitHub Pages.

  3. V sekci Source vyber main branch a klikni na Save.

  4. Po chvíli by se měla objevit URL, kde je tvůj web hostován, například:

    arduino
    https://tvoje-uzivatelske-jmeno.github.io/hyx-website/

Krok 5: Navštívení tvých webových stránek
